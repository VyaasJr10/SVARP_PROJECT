SVARP - Smart Voice Assisted Relief Platform

An Android health assistant app built in Java that helps users assess their symptoms and get personalised health guidance, completely offline and free to use.

a] Features
- 🎙️ Voice + tap symptom input
- 🟢 SAFE / MODERATE / DANGER risk assessment
- 🌍 UI in 7 Indian languages
- 📵 Fully offline — no internet needed
- 🚨 Emergency 108 button
- 🔔 Daily health tip notifications
- 📋 Detailed symptom information

b] Tech Stack
- Java + Android Studio
- Android SpeechRecognizer API
- AlarmManager + BroadcastReceiver
- SharedPreferences
- Material Design Components

c] Note
This app is for informational purposes only and does not replace professional medical advice.
